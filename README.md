# Кадровое агентство (AgencyApp)

Приложение для управления базой данных кадрового агентства с использованием Windows Forms, Entity Framework Core и SQLite.

## Возможности

- **Соискатели** - добавление, редактирование, удаление соискателей с проверкой возраста (>= 14 лет)
- **Вакансии** - управление вакансиями работодателей
- **Резюме** - управление резюме соискателей
- **Навыки** - справочник навыков
- **Работодатели** - управление компаниями-работодателями
- **Собеседования** - планирование и отслеживание собеседований
- **Подбор кандидатов** - поиск кандидатов по навыкам вакансии и поиск вакансий по навыкам

## Требования

- .NET 8.0
- Visual Studio 2022
- Windows 10/11

## Запуск

### Вариант 1: Запуск из Visual Studio

1. Откройте файл `AgencyApp.sln` в Visual Studio 2022
2. Нажмите `F5` или кнопку "Запуск" (▶️)

### Вариант 2: Запуск скомпилированного приложения

1. Запустите файл `Запустить.bat`
   ```cmd
   Запустить.bat
   ```

### Вариант 3: Ручной запуск

1. Перейдите в папку с проектом
2. Выполните сборку:
   ```cmd
   dotnet build -c Release
   ```
3. Запустите:
   ```cmd
   dotnet run --project AgencyApp.csproj
   ```

## Тестирование функционала

### Соискатели

**Добавление:**
1. Заполните поле "ФИО"
2. Заполните поле "Контакты"
3. Выберите дату рождения
4. Нажмите "Добавить"

> ⚠️ Если возраст соискателя менее 14 лет, появится сообщение об ошибке

**Редактирование:**
1. Выберите соискателя в списке
2. Измените данные в полях ввода
3. Нажмите "Изменить"

**Удаление:**
1. Выберите соискателя в списке
2. Нажмите "Удалить"
3. Подтвердите удаление в диалоговом окне

### Подбор кандидатов

**Поиск кандидатов по вакансии:**
1. Выберите вакансию из списка
2. Нажмите "Найти кандидатов"
3. Отобразятся кандидаты с подходящими навыками

> Если кандидаты не найдены, появится сообщение "Кандидаты не найдены"

**Поиск вакансий по навыку:**
1. Выберите навык из списка
2. Нажмите "Найти вакансии"
3. Отобразятся вакансии с указанным навыком

## Первоначальные данные

При первом запуске автоматически создаётся база данных SQLite и заполняется тестовыми данными:
- 3 соискателя
- 2 работодателя
- 4 вакансии
- 3 резюме
- 5 навыков
- Несколько собеседований
